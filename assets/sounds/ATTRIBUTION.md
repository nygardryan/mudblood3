# Sound attribution

Curated open-source sound effects used by Mud & Blood. See `assets/sounds/*.ogg`.

## CC0 (public domain — no attribution required)

**Free Firearm Sound Library — Prepared SFX**  
https://opengameart.org/content/the-free-firearm-sound-library  
Recorded gunfire (Springfield 1917, Mosin Nagant, Colt 1911, PPSh-41, Winchester Model 12, and others). Trimmed for in-game use.

**Kenney — Sci-Fi Sounds**  
https://opengameart.org/content/sci-fi-sounds  
Explosions, engines, vehicle brakes, metal impacts.

**Guialgarve — Flamethrower (Freesound, CC0)**  
https://freesound.org/people/Guialgarve/sounds/572395/  
Flamethrower spray (`flame_1.ogg`).

**AslakHostaker — Flamethrower bursts (Freesound, CC0)**  
https://freesound.org/people/AslakHostaker/sounds/395038/  
Flamethrower spray (`flame_2.ogg`).

**Kenney — Interface Sounds**  
https://opengameart.org/content/interface-sounds  
UI clicks, errors, alarms, event stingers.

**Trumpet fanfare**  
https://opengameart.org/content/trumpet-fanfare  
Promotion stinger (`promote_1.ogg`).

**Classic fanfare lick**  
https://opengameart.org/content/classic-fanfare-lick  
Promotion stinger (`promote_2.ogg`).

**Zagge28 — Incoming mortar whistles (Freesound)**  
https://freesound.org/people/Zagge28/  
Incoming shell warning (`whistle_1.ogg`–`whistle_3.ogg`).

**qubodup — Male pain/vocal sounds**  
https://opengameart.org/content/15-vocal-male-strainhurtpainjump-sounds  
Explosion casualty screams (`scream_1.ogg`, `scream_2.ogg`).

**MrFossy — Pain grunts (Freesound, CC0)**  
https://freesound.org/people/MrFossy/sounds/547209/  
Explosion casualty scream (`scream_3.ogg`).

**Craig Smith — Aircraft flybys (Freesound, CC0)**  
https://freesound.org/people/craigsmith/sounds/437795/  
https://freesound.org/people/craigsmith/sounds/675744/  
Prop engine loops (`plane_1.ogg`, `plane_2.ogg`) and flyby passes (`plane_flyby_1.ogg`, `plane_flyby_2.ogg`).
